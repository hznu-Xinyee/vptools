# VP 项目运维命令文档

## 快速重启命令

### 本地开发环境

#### 前端重启
```bash
# 杀掉现有进程
ps aux | grep "vite" | grep -v grep | awk '{print $2}' | xargs kill -9

# 重新启动
cd /Users/montageai/Desktop/vp_dev/vptools/frontend
npm run dev
```

#### 后端重启
```bash
# 杀掉现有进程
ps aux | grep "uvicorn main:app" | grep -v grep | awk '{print $2}' | xargs kill -9

# 重新启动
cd /Users/montageai/Desktop/vp_dev/vptools/backend
source venv/bin/activate
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

#### 视频翻译服务重启
```bash
# 杀掉现有进程
ps aux | grep "uvicorn app.main:app" | grep -v grep | awk '{print $2}' | xargs kill -9

# 重新启动
cd /Users/montageai/Desktop/vp_dev/vptools/video-translation-docker-service
source venv/bin/activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8080
```

### 生产环境（服务器）

#### 前端重启
```bash
cd /opt/VP/frontend
npm run build
sudo systemctl reload nginx
```

#### 后端重启
```bash
# 使用 tmux（推荐）
tmux kill-session -t backend
tmux new -s backend -c /opt/VP/backend 'venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000'

# 传统方式
ps aux | grep "uvicorn main:app" | grep -v grep | awk '{print $2}' | xargs kill -9
cd /opt/VP/backend && nohup venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000 > backend.log 2>&1 &
```

#### 视频翻译服务重启
```bash
# 使用 tmux（推荐）
tmux kill-session -t docker-service
tmux new -s docker-service -c /opt/VP/video-translation-docker-service 'venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080'

# 传统方式
ps aux | grep "uvicorn app.main:app" | grep -v grep | awk '{print $2}' | xargs kill -9
cd /opt/VP/video-translation-docker-service && nohup venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080 > docker_service.log 2>&1 &
```

## 服务管理

### 使用 tmux 管理服务（推荐，SSH 断开后服务不会停止）

#### 安装 tmux
```bash
sudo apt install tmux -y
```

#### 后端服务
```bash
# 创建 tmux 会话并启动后端服务
tmux new -s backend -c /opt/VP/backend 'venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000'

# 断开会话（服务继续运行）
# 在 tmux 会话中按: Ctrl+B 然后按 D

# 重新连接到后端服务会话
tmux attach -t backend

# 停止后端服务
# 在 tmux 会话中按: Ctrl+C

# 查看所有 tmux 会话
tmux ls

# 删除 tmux 会话
tmux kill-session -t backend
```

#### Docker 服务（视频翻译）
```bash
# 创建 tmux 会话并启动 Docker 服务
tmux new -s docker-service -c /opt/VP/video-translation-docker-service 'venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080'

# 断开会话（服务继续运行）
# 在 tmux 会话中按: Ctrl+B 然后按 D

# 重新连接到 Docker 服务会话
tmux attach -t docker-service

# 停止 Docker 服务
# 在 tmux 会话中按: Ctrl+C

# 删除 tmux 会话
tmux kill-session -t docker-service
```

### 传统方式（SSH 断开后服务会停止，不推荐）

#### 后端服务
```bash
# 启动后端服务（前台）
cd /opt/VP/backend
venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000

# 后台启动后端服务（使用 nohup）
cd /opt/VP/backend
nohup venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000 > backend.log 2>&1 &

# 停止后端服务
ps aux | grep uvicorn
kill <PID>

# 查看后端日志
tail -f /opt/VP/backend/backend.log

# 测试后端健康检查
curl http://localhost:8000/health
```

#### Docker 服务（视频翻译）
```bash
# 启动 Docker 服务（前台）
cd /opt/VP/video-translation-docker-service
venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080

# 后台启动 Docker 服务（使用 nohup）
cd /opt/VP/video-translation-docker-service
nohup venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080 > docker_service.log 2>&1 &

# 停止 Docker 服务

kill <PID>

# 测试 Docker 服务
curl http://localhost:8080/
```

### PostgreSQL 数据库
```bash
# 启动 PostgreSQL
sudo systemctl start postgresql

# 停止 PostgreSQL
sudo systemctl stop postgresql

# 重启 PostgreSQL
sudo systemctl restart postgresql

# 查看状态
sudo systemctl status postgresql

# 连接数据库
sudo -u postgres psql

# 创建数据库
sudo -u postgres psql -c "CREATE DATABASE vp_db;"

# 创建用户
sudo -u postgres psql -c "CREATE USER montageai WITH PASSWORD 'montageai';"

# 授权
sudo -u postgres psql -c "GRANT ALL PRIVILEGES ON DATABASE vp_db TO montageai;"
```

## 快速重启所有服务

```bash
# 重启后端服务
ps aux | grep "uvicorn main:app" | grep -v grep | awk '{print $2}' | xargs kill -9
cd /opt/VP/backend && nohup venv/bin/uvicorn main:app --host 0.0.0.0 --port 8000 > backend.log 2>&1 &

# 重启 Docker 服务
ps aux | grep "uvicorn app.main:app" | grep -v grep | awk '{print $2}' | xargs kill -9
cd /opt/VP/video-translation-docker-service && nohup venv/bin/uvicorn app.main:app --host 0.0.0.0 --port 8080 > docker_service.log 2>&1 &

# 重启 Nginx
sudo systemctl reload nginx

# 重启 PostgreSQL
sudo systemctl restart postgresql
```

## 修改配置后生效

修改以下文件后需要重启对应服务：

1. **后端代码修改** (`/opt/VP/backend/app/`): 重启后端服务
2. **前端代码修改** (`/opt/VP/frontend/src/`): 重新构建前端 `npm run build`
3. **Nginx 配置修改**: `sudo nginx -t && sudo systemctl reload nginx`
4. **环境变量修改** (`.env`): 重启对应服务
5. **字幕位置参数修改**: 重启后端服务
